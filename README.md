# Another Snake Game
Game installation:<br>
<ol>
<li><code>git clone git@github.com:un-ch/another_snake_game.git</code></li>
<li><code>cd another_snake_game</code></li>
<li><code>make</code></li>
</ol>
Launch the game:<br>
<code>make run</code></li><br>
